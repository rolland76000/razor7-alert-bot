# razor7-alert-bot

Bot minimal qui envoie un message Telegram dès que Razor7 publie un tip sur Oddspedia.

## Setup

1. Ajouter TELEGRAM_BOT_TOKEN et TELEGRAM_CHAT_ID sur Render (entrer ces valeurs dans les variables d’environnement).
2. Déployer avec render.yaml.
3. Le bot tourne toutes les 60 secondes.
